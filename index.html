  • i) Be careful with brackets(). Don't write (f(x)D^n), (f(x)y^n), sinx, cosx, logx, rootx, e^x, or a mixture of these, write (f(x))D^n, (f(x))y, f(x)y, f(x)D^n, sin(x), cos(x), log(x), root(x) & e^(x). Here, f(x) is any function

  • ii) Don't write 'D^n = y/f(x)', write 'D^n = (1/f(x))y', because 'D^n' & 'y' terms can't be divided. So, make sure to simplify the equation in f(x)D^n+...=g(x) format

  • iii) Don't write rooty or root(y), write y^0.5. Don't write 1/y, write y^-1. Don't write nf(x), write n*f(x). And remember that, D = dy/dx ('y' is the dependent variable, and 'x' is the independent variable.)

  • iv) So, always give the co-efficients in front of D^n or y (Don't write y*f(x), write f(x)*y) & 'y' can't be a power of f(x (f(x)^y is not allowed)
